# C语言指针

## 核心概念

指针是保存地址的变量。普通变量保存具体值，指针变量保存另一个变量在内存中的地址。

理解指针时可以抓住两件事：`&` 取地址，`*` 解引用。

## 基本写法

```c
int a = 10;
int *p = &a;
```

这里 `p` 保存变量 `a` 的地址。

## 取地址和解引用

```c
printf("%p", p);
printf("%d", *p);
```

`p` 表示地址，`*p` 表示这个地址上存放的值。

## 通过指针修改变量

```c
int a = 10;
int *p = &a;
*p = 20;
```

执行后，变量 `a` 的值会变成 `20`。

## 指针和数组

数组名在很多表达式中会退化为首元素地址。

```c
int arr[3] = {1, 2, 3};
int *p = arr;
```

此时 `p` 指向 `arr[0]`。

## 指针作为函数参数

如果希望函数修改外部变量，可以传入变量地址。

```c
void change(int *p) {
    *p = 100;
}
```

调用时：

```c
int x = 1;
change(&x);
```

## 空指针

空指针通常写作 `NULL`，表示这个指针暂时不指向有效地址。

```c
int *p = NULL;
```

空指针不能直接解引用。

## 常见误区

未初始化指针不能直接使用。野指针、空指针解引用都会导致程序行为不可预测。

另一个常见误区是把地址和值混在一起：`p` 是地址，`*p` 才是地址里的值。

## 学习建议

初学指针时，先画内存图：变量在哪，指针保存哪个地址，`*p` 访问的是哪个值。
