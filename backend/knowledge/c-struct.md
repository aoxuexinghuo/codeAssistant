# C语言结构体

## 核心概念

结构体是 C 语言中的自定义数据类型，用来把多个不同类型的数据组合成一个整体。它适合描述具有多个属性的数据对象，例如学生、坐标、图书、链表节点。

结构体解决的问题是：当一个对象需要同时保存多个字段时，不必使用多个分散变量，而是可以把这些字段组织到同一个类型中。

## 基本定义

```c
struct Student {
    int age;
    char name[20];
};
```

这段代码只是定义了一个结构体类型，还没有创建具体变量。

## 创建变量

```c
struct Student s1;
s1.age = 18;
```

访问结构体变量的成员使用点运算符 `.`。

## 定义时创建变量

```c
struct Student {
    int age;
    char name[20];
} s1;
```

这种写法会在定义结构体类型的同时创建变量 `s1`。

## 使用 typedef 简化写法

```c
typedef struct {
    int age;
    char name[20];
} Student;

Student s1;
```

使用 `typedef` 后，声明变量时不需要每次都写 `struct Student`。

## 结构体初始化

```c
struct Student s1 = {18, "Tom"};
```

初始化顺序要和成员定义顺序一致。

## 结构体指针

```c
struct Student *p = &s1;
printf("%d", p->age);
```

结构体指针访问成员使用箭头运算符 `->`，它等价于 `(*p).age`。

## 常见误区

定义结构体类型不等于创建变量。`struct Student { ... };` 只是定义类型，真正使用前还需要声明变量。

另一个常见误区是混淆 `.` 和 `->`：结构体变量用 `.`，结构体指针用 `->`。

## 适用场景

结构体常用于保存一组相关数据，例如学生信息、坐标点、日期、链表节点、二叉树节点等。
